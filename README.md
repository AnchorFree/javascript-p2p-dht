javascript-p2p
==============

Experiment for building a JavaScript-based DHT for storage.

Instructions
------------
1. Launch tracker.htm in a browser (Chrome 32+ or FF 29+)
1. Launch one or more copies of peer.htm in a browser (Chrome 32+ or FF 29+)
